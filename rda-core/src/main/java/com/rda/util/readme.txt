util包中的工具类主要开发目的为:
1.代理分散在各个不同jar中的现成工具类,使soofa使用者可以在统一的org.soofa.util包
	下找到其想用的绝大部分工具类
2.扩展现有的工具类
3.在找不到现成工具类的情况下,开发新的工具类
